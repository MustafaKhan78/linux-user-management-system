# Interactive User Management System

A Bash Shell Scripting project for Linux user administration.

## Features

- Add User
- Delete User
- Lock User
- Unlock User
- List Users
- Activity Logging
- Colored Terminal Output
- Menu Driven Interface

## Technologies

- Bash
- Linux User Management Commands
- awk
- passwd
- useradd
- userdel

## How to Run

chmod +x user_management.sh
./user_management.sh
